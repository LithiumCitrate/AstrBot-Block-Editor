🧩 欢迎使用 Block Editor
通过拖拽积木块，零代码创建 AstrBot 插件
1
选择触发器
从左侧拖拽一个「触发器」块（绿色）到工作区，决定插件何时触发
2
添加动作
拖拽「动作」块（紫色）到触发器下方，自动连接。点击块可修改参数
3
编译导出
点击「编译预览」查看代码，「导出」生成插件目录
